# The Founder’s Journal

A local AI-inspired self-awareness and productivity tracking system designed to help founders track daily execution, emotional patterns, habits, deep work, and priority alignment.

## Overview

The Founder’s Journal is a private browser-based journaling dashboard built with Python, HTML, JavaScript, and Chart.js. It helps users move beyond simple journaling by turning daily reflection into measurable analytics.

The app focuses on one main question:
**Are my actions matching my stated priorities?**

It tracks morning intentions, evening reflections, revenue hours, deep work, habits, emotional triggers, escape patterns, and repeated behavioral loops.

## Features

* Morning and evening journaling framework
* Daily priority and regret-based task tracking
* Habit tracking for gym, reading, outreach, sleep, clean eating, and discipline
* Metrics for revenue hours, deep work, wake time, sleep time, and shipping output
* Pattern detection for distractions, avoidance, and emotional triggers
* Analytics dashboard using Chart.js
* History view for past entries
* Local JSON storage
* CSV and JSON export
* Runs privately on localhost

## Tech Stack

* Python
* HTML
* CSS
* JavaScript
* Chart.js
* Local JSON storage
* Python HTTP server

## Why I Built This

I built this project to create a personal operating system for self-awareness, discipline, and founder-level accountability. Instead of only writing thoughts, the tool helps identify behavior patterns, measure execution, and create rules for better decision-making the next day.

## How to Run

```bash
python3 founder_journal_v2.py
```

Then open:

```text
http://localhost:8765
```

The app stores data locally at:

```text
~/founder_journal_data/entries_v2.json
```

## Project Goal

The goal of this project is to combine journaling, behavior tracking, and analytics into a simple local tool that helps users understand how they spend time, where they avoid hard tasks, and how consistently they execute against their priorities.

## Future Improvements

* AI-generated daily insights
* Weekly summary reports
* Pattern prediction based on past behavior
* Goal recommendation engine
* Cloud sync option
* Authentication and multi-user support
* Mobile-friendly PWA version
